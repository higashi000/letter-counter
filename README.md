# letter-counter

Count number of characters in your text!<br>
[count.higashi.dev](https://count.higashi.dev)<br>

## Try your localhost
```
$ git clone https://github.com/higashi000/letter-counter

$ cd letter-counter

$ go build

$ ./letter-counter
```

After, please access to [localhost:5000](http://localhost:5000).<br>
